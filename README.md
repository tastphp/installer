# tastphp-installer

tastphp installer tool

## Requirement

* [composer](https://getcomposer.org/)
* Zip PHP extension

## Installation

```
git clone https://github.com/tastphp/tastphp-installer.git

cd tastphp-installer

composer update

chmod +x tastphp
```

## usage

### check version
```
$ php tastphp -V
tastphp-installer v1.0.0
```

### install tastphp
```
$ php tastphp install
```
